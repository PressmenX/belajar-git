## 🚀 Cara Menggunakan Repo Ini

### Prasyarat
- Git versi 2.x ke atas
- Akun GitHub
- Terminal (Git Bash / macOS Terminal / Linux)

### Clone Repo
```bash
git clone git@github.com:PressmenX/belajar-git.git
cd belajar-git
```

### Lihat History Belajar
```bash
git log --oneline --graph
```
